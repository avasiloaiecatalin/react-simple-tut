## Adaugarea package-urilor pentru styling

Pentru stilizare, am putea folosi tool-uri basic, precum CSS, dar exista alternative mult mai solide si cu un design placut, motiv pentru care vom introduce shadcn/ui, impreuna cu TailwindCSS.

### Ce e shadcn/ui?

Shadcn/ui este o colectie de componente React (input-uri, butoane, etc.) care pot fi refolosite la nevoie in crearea de pagini pentru aplicatia noastra.

### Ce este TailwindCSS?

TailwindCSS este un <Link href="https://www.codecademy.com/resources/blog/what-is-a-framework/">framework</Link> care pune la dispozitie un set de clase CSS bine realizate si structurate.
