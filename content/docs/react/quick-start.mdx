## Crearea proiectului

Primul pas pentru a incepe dezvoltarea folosind React JS este crearea unui proiect.

import { DynamicCodeBlock } from 'fumadocs-ui/components/dynamic-codeblock';

<DynamicCodeBlock
  lang='js'
  code={'npm create vite@latest'}
/>

Dupa executarea comenzii intr-un terminal se va crea un proiect simplu, pe care il vom folosi ca baza pentru ce urmeaza sa construim.

## Adaugarea unui package pentru routing management

Ne dorim sa putem avea continutul distribuit in pagini diferite, de aceea avem nevoie de un package care sa se ocupe de managementul rutelor in cadrul proiectului.

<DynamicCodeBlock
  lang='js'
  code={'npm i react-router'}
/>
