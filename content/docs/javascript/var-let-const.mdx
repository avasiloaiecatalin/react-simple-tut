## var

`var` e considerat global doar daca nu e declarat in interiorul unei functii.

<DynamicCodeBlock
  lang='js'
  code='var a = "hi";
    function newFunction() {
        var b = "hello";
    }
    console.log(b); // error: b is not defined'
/>

`var` poate fi redeclarat / modificat:

<DynamicCodeBlock
  lang='js'
  code='var a = "hi";
    var a = "hello";
    //
    var a = "hey hi";
    a = "hello";'
/>

## let

<DynamicCodeBlock
  lang='js'
  code='let a = "hi";
  let x = 4;

if (x > 3) {
let b = "hello";
console.log(b);// "hello"
}

console.log(b) // b is not defined'
/>

`let` poate fi modificat, dar nu redeclarat:

<DynamicCodeBlock
  lang='js'
  code='let a = "hi";
    a = "hello";'
/>

<DynamicCodeBlock
  lang='js'
  code='let a = "hi";
    let a = "hello"; // error: Identifier a has already been declared'
/>

Totusi, daca il declari in blocuri diferite, e ok:

<DynamicCodeBlock
  lang='js'
  code='let a = "hi";
    if (true) {
        let a = "hello";
        console.log(a); // "hello"
    }
    console.log(a); // "hi"'
/>
